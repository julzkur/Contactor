# Contactor
# Assignment 2 MAPP

## Description

App for displaying a list of saved contacts with their details.



## Running the app

Open terminal and navigate into the root directory

`cd contactor/contactor`

Install all necessary dependencies

`npm install`

Run the application

`npx expo start`

If you're running on a Simulator/Emulator

    iOS: `npx expo start --ios`
    android: `npx expo start --android`
